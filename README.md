# electron-vesktop

This package is a meta package that provides the latest available stable Electron build for Vesktop on Arch Linux. It simply depends on the most recent `electronXX-vesktop` package, ensuring you always have the latest version installed.

For more information about the custom Electron build, the patches applied, and the reasons behind them, see the [electron36-vesktop](https://github.com/Lungooo/electron36-vesktop/README.md) repository.
